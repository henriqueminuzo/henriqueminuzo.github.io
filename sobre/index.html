Sobre

Uma caixa de fósforos é um objeto simples, mas não pobre.

Ela tem estrutura, forma e finalidade. Há a madeira que sustenta, o invólucro que protege, o rótulo que identifica e a gaveta que guarda os palitos alinhados, prontos.

Uma caixinha pequena costuma trazer quarenta fósforos. Nem poucos, nem excessivos. Cada um existe para um único gesto, breve e decisivo.

Este espaço foi pensado assim.

Cada texto é um fósforo. Não se escreve para acumular, mas para usar. Alguns acendem luz, outros apenas calor. Alguns falham. Faz parte.

Quando os fósforos acabarem, acabam. E se um dia todos forem gastos com honestidade, talvez seja hora de aceitar o silêncio (ou, quem sabe, abrir uma nova caixa).

Aqui a escrita é lenta. A fé não é reinventada. A tradição não é ornamento, mas estrutura.

Nada foi feito para brilhar sozinho. Tudo foi feito para servir ao momento certo de ser riscado.
